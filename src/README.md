# CO-COOK RECIPE

An interactive web application for creating recipes by combining ingredients. Features a drag-and-drop interface with a MoMA-inspired minimalist design.

## Features

- 🍳 **Interactive Cooking Experience** - Drag ingredients into a pan to create recipes
- 👥 **Co-Cook Mode** - Two roommates' ingredients on left and right sides
- 📱 **Responsive Design** - Works on desktop and mobile devices
- 🎨 **Swiss Design Aesthetic** - Clean, minimalist interface with pink theme (#EF67A7)
- 🔤 **Custom Typography** - Swiss Grit and Mundial Narrow Variable fonts
- 📋 **Fridge Tracker** - Track forgotten foods and get recipe suggestions
- 🎉 **Visual Feedback** - Fireworks and animations when recipes are created
- 📱 **QR Code Sharing** - Share the app via QR code

## Pages

1. **Home Page** - Main landing page with navigation
2. **Co-Cook Recipe** - Interactive cooking interface
3. **Fridge Detail** - Track and manage forgotten foods
4. **QR Code** - Share the app

## Tech Stack

- **React** - UI framework
- **TypeScript** - Type safety
- **Tailwind CSS v4** - Styling
- **React DnD** - Drag and drop functionality
- **Lucide React** - Icons
- **QR Code React** - QR code generation

## Custom Fonts

- **Swiss Grit** - Used for all headings and key UI elements
- **Mundial Narrow Variable** - Used for body text
